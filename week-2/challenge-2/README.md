# Challenge 2 - 주민번호로 지역 판별하기

## 주민번호 뒷자리 정보

주민번호는 사진과 같이 구성되어 있습니다.

![&#xC8FC;&#xBBFC;&#xBC88;&#xD638; &#xC758;&#xBBF8;](../../.gitbook/assets/image%20%286%29.png)

우리는 공개된 지역코드를 가지고 지역을 판별하도록 하겠습니다.

## 지역 코드

지역 코드는 아래와 같습니다.

```text
서울특별시 : 00 - 08
부산광역시 : 09 - 12
인천광역시 : 13 - 15
경기도 : 16 - 25
강원도 : 26 - 34
충청북도 : 35 - 39
대전광역시 : 40
충청남도 : 41 - 43, 45 - 47
세종특별자치시 :44, 96
전라북도 : 48 - 54
전라남도 : 55 - 66
광주광역시 : 55, 56
대구광역시 : 67 - 70
경상북도 : 71 - 81
경상남도 : 82 - 84, 86 - 90
울산광역시 : 85
제주특별자치도 : 91 - 95
```
