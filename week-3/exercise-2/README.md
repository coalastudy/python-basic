# 자율과제 2 - 가위바위보 게임 반복하기

## 임의의 난수 발생시키기

두번째 플레이어를 컴퓨터라고 가정하고, 입력받는 것이 아니라 임의의 난수를 입력받습니다.

## 사용자가 이길 때까지 반복하기

조건문을 통하여 사용자가 이길 때 게임을 종료시킵니다.
