# Challenge 2 - 비만도 계산기 만들기

## 입력받기

 BMI를 계산하기 위하여 키와 몸무게를 사용자로부터 입력받고, 부가 정보로 성별, 나이를 입력받으시오.

## 계산하기

{% hint style="info" %}
BMI = 몸무게 / \(키 \* 키 / 10000\)
{% endhint %}
